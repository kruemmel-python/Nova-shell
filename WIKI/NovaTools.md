# Nova Tools

## Rolle

Tools sind benannte Operationen mit klaren Seiteneffekten oder Datenresultaten.

## Typen

- Built-in Runtime Tools
- Backend Operations
- Service- und Package-Werkzeuge
- Event- und State-Werkzeuge

## Beispiele

- `rss.fetch`
- `atheria.embed`
- `system.log`
- `event.emit`
- `flow.run`
- `state.set`
- `state.get`
