# Glossary

| Begriff | Bedeutung |
| --- | --- |
| Agent | modellgestuetzte Laufzeitinstanz |
| Dataset | strukturierte Eingabemenge |
| Flow | deklarativer Ablauf |
| Tool | benannte Operation |
| Event | Signal fuer Automation |
| Service | laufender Dienst in der Plattform |
| Package | installierbares Artefakt |
| Mesh | verteilte Worker-Schicht |
| Control Plane | Queue-, Schedule- und API-Ebene |
| Traffic Plane | Routing- und Health-Schicht |
| Lockfile | reproduzierbare Modulaufloesung |
