# Tutorial: Technology Radar

## Ziel

Ein einfacher Radar-Workflow sammelt Signale und laesst sie durch einen Agenten verdichten.

## Schritte

1. Dataset definieren
2. Agent definieren
3. Flow anlegen
4. `ns.run` ausfuehren

## Ergebnis

Ein strukturierter Radar-Output fuer weitere Analyse oder Reports.
