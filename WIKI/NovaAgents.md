# Nova Agents

## Rolle

Agenten sind modellgestuetzte Aufgabeninstanzen innerhalb der Runtime.

## Eigenschaften

- Provider- und Modellauswahl
- Prompt-Versionen
- Governance
- Tool-Sandbox
- Memory-Sharding
- Evaluationen

## Ausfuehrung

```text
Task
  ↓
Governance
  ↓
Prompt Resolution
  ↓
Provider Invocation
  ↓
Memory Write
  ↓
Evaluation
```
