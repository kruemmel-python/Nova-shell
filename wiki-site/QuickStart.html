Quick Start

Ziel

Innerhalb weniger Minuten:

  • Shell starten
  • ersten Befehl ausfuehren
  • erstes .ns-Programm laufen lassen
  • Plattformstatus abrufen

1. Shell starten

python -m nova_shell

oder nach Installation:

nova-shell

2. Erster Befehl

nova> py 1 + 1
2

3. Erste Nova-Datei

agent helper {
  model: local
}

dataset notes {
  items: [{text: "hello nova"}]
}

flow boot {
  helper summarize notes -> result
}

Ausfuehrung:

nova> ns.run hello.ns

4. Status

nova> ns.status
