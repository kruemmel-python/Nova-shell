# Media Reports

Dieser Ordner sammelt die publizierbaren Manifest-Fassungen zu Nova-shell.

Enthalten sind:

- `MANIFEST_2026_FINAL.md`
- `Minfest_2026.md`
- `Manifest_2026_OpEd_EN.md`
- `Manifest_2026_Pitch_DE.md`
- `Manifest_2026_Pitch_EN.md`
- `Manifest_2026_Journalistic_DE.md`
- `Manifest_2026_Journalistic_EN.md`

Empfohlene Nutzung:

- `Pitch`: fuer LinkedIn, Newsletter, Intro-Seiten, Medium-Posts
- `Journalistic`: fuer neutralere Magazin- oder Reportageformate
- `OpEd`: fuer meinungsstarke internationale Publikationen
- `FINAL` / `Minfest`: fuer laengere Grundsatzfassungen
